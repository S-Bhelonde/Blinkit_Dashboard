# 📊 Blinkit Sales Dashboard (Power BI)

This project presents an interactive Blinkit Sales Dashboard designed to analyze and visualize key business metrics for better decision-making. The dashboard provides a comprehensive overview of sales performance, item distribution, outlet analysis, and customer ratings.

🔍 Key Features
Performance Overview
Total Sales: $1.20M
Average Sales: $141
Average Rating: 3.92
Number of Items: 9K
Sales Analysis
Sales breakdown by Outlet Size (Small, Medium, High)
Sales comparison by Outlet Location Type (Tier 1, Tier 2, Tier 3)
Product Insights
Item type-wise sales distribution (Fruits, Snacks, Dairy, etc.)
Fat content analysis (Low Fat vs Regular)
Outlet Performance
Sales, number of items, average sales, and ratings by outlet type
Comparison across Supermarket Types and Grocery Stores
Trend Analysis
Year-wise outlet establishment and sales trends
Interactive Filters
Outlet Location Type
Item Type
Outlet Size

🎯 Objective

The main objective of this dashboard is to:

Analyze sales trends and performance
Identify high-performing products and outlets
Support data-driven business decisions

🛠️ Tools Used
Power BI
Data Visualization Techniques
Data Cleaning & Transformation

📌 Use Case
This dashboard can be used by:

- Business Analysts
- Retail Managers
- Sales Teams

to monitor performance, optimize inventory, and improve overall sales strategy.
